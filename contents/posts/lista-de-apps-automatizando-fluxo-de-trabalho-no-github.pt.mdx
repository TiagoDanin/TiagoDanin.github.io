Uma das melhores coisa de usar o GitHub é a quantidade de integrações disponíveis, hoje eu separei alguma das que eu mais uso:

* Travis CI: Testes & Build
* Crowdin: Tradução
* ImgBot: Otimização de Imagens
* Codacy: Analise de Código
* GitHub (Telegram): Notificação de Issues/PR's & Commits
* Renovate: Atualização de Dependências
* PR Triage: Organização de Issues/PR's
* Snyk: Verificação de Vulnerabilidade
* CuboHub: Gerado de Página Estáticas
* WIP: Proteção de Branch & Organização de PR's
