One of the best things about using GitHub is the amount of integrations available. Today I've separated some of the ones I use most:

* Travis CI: Testing & Build
* Crowdin: Translation
* ImgBot: Image Optimization
* Codacy: Code Analysis
* GitHub (Telegram): Issue/PR & Commit Notifications
* Renovate: Dependency Updates
* PR Triage: Issue/PR Organization
* Snyk: Vulnerability Checking
* CuboHub: Static Page Generator
* WIP: Branch Protection & PR Organization
